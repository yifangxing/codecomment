本文件夹的作用是从生成的结果中提取出生成的注释和参考的注释，并且只提取那些方法中有孩子节点注释的方法的注释为评估数据（即只有这些方法在注释的生成过程中用到了我们注释生成的思路）。

process文件为从生成的结果中提取出生成的注释和参考的注释，并且所有生成的注释和参考的注释都只提取第一句话。
test文件为从生成的结果中提取出生成的注释和参考的注释，并且所有生成的注释和参考的注释都提取全部。
full_level：按照整个调用图的层级提取注释，在调用图中有多少层级就划分多少层。
level: 将整个调用图节点划分为4个部分，调用图第一层不参与划分，调用图第二层一个部分，剩余层数从高层到低层依次等量划分为3个部分。
